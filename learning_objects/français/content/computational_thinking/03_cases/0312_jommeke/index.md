# Coloriser Jommeke
**Un système d'IA pour coloriser des bandes dessinées de Jommeke**

Sur le site web de dwengo, vous trouverez un système d'IA pour coloriser automatiquement des bandes dessinées de Jommeke. Il s'agit d'un système d'IA générative qui peut être utilisé pour donner des couleurs aux premiers Jommeke, qui étaient encore en noir et blanc. Cependant, cette colorisation ne se déroule pas toujours parfaitement.

Ce cas vous donne un meilleur aperçu de ce que l'IA peut apporter à d'autres domaines (comme l'art), mais aussi de ses limites. Consultez également le [thème de cours 'Art'](https://www.dwengo.org/art/) dont est issu ce cas.
