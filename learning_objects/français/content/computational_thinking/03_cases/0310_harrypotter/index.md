# Conversations dans les livres de Harry Potter

Si, en cours de néerlandais (ou d’une autre langue), tu veux extraire des informations d’un livre, tu peux visualiser de manière structurée des données issues du texte en utilisant des graphes. Les graphes permettent par exemple de mettre en évidence les relations entre les personnages d’un livre. Dans ce cas, nous avons visualisé les conversations du livre *Harry Potter à l’école des sorciers*, d’abord uniquement pour le chapitre 11 (figure 1), puis pour l’ensemble du livre (figure 2).
