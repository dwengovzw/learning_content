# La tâche/la problématique

**La tâche/la problématique est fonctionnellement pertinente pour le groupe cible.**
De préférence, la tâche est liée à un objectif final/minimum comportant des contenus spécifiques à la filière d’études. Par exemple, l’élaboration d’un plan de culture s’inscrit dans le domaine d’application 'biotechniek' pour le cours de technologie dans le premier degré, filière B.

**La tâche/la problématique aborde un problème/une question socialement pertinent(e).**

**La tâche/la problématique met les élèves au défi de mobiliser les concepts de base afin d’aboutir à une solution.**

**La tâche peut mener à l’utilisation d’un ordinateur pour résoudre le problème.**
