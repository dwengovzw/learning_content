# Abstraction

*Les élèves laissent de côté, lors de l’analyse de la tâche/de l’énoncé du problème, les informations non pertinentes pour résoudre le problème.*
  
**Exemple**<br>
Une institutrice de maternelle doit veiller à ce que tous les enfants aient mis et fermé leur veste à temps.

Ce qui est pertinent ici, c’est uniquement de savoir si la veste s’ouvre/se ferme à l’avant, si elle est entièrement ouverte/fermée et quel type de fermeture elle possède. La couleur, la coupe, avec ou sans capuche, le type de textile de la veste… sont autant d’informations non pertinentes.

ET/OU

*Les élèves représentent de manière abstraite une description issue de l’énoncé du problème.*

**Exemple**<br>
L’énoncé du problème consiste à établir un plan de potager pour quatre types de légumes. 

Pour la résolution, il est préférable de représenter le potager de façon abstraite par une grille. Les types de légumes devront également être représentés d’une certaine manière abstraite.

ET/OU

*Les élèves doivent travailler avec une représentation/description abstraite dans la tâche.*

**Exemple 1**<br>
Dans la réalité, les agriculteurs souhaitent parfois élaborer un plan de fertilisation en vue d’optimiser l’efficacité pour l’année suivante. Cela signifie qu’ils utilisent des images satellite afin de ne pas appliquer une fertilisation uniforme sur l’ensemble de leur champ. Un pixel sur une image satellite représente alors une surface d’environ 5 mètres sur 5 mètres, et les agriculteurs doivent travailler à partir de cette donnée abstraite.
