# Grille d'observation

Cette grille d’observation peut être utilisée pour concevoir vous-même une tâche ou pour comprendre et/ou évaluer dans quelle mesure la pensée computationnelle est (ou peut être) abordée. 


* Partez du schéma des quatre concepts de base.
 
* Utilisez les précisions/points d’attention relatives aux concepts de base pour revoir votre tâche ou pour interpréter, évaluer et, si nécessaire, ajuster le travail d’autrui ou des élèves. 

* Cochez le cas échéant.

<div class="alert alert-box alert-danger">
<strong><h5>Attention</h5></strong> 
<ul>
    <li>Les quatre concepts de base ne doivent pas nécessairement tous être abordés (cela dépend de l’énoncé du problème et de sa complexité).</li>
    <li>Les concepts de base sont mobilisés en vue de résoudre un défi/problème et ne sont pas travaillés uniquement de manière isolée.</li>
    <li>Des approches imbriquées sont également possibles et parfois même nécessaires (p. ex., pour des sous-problèmes/sous-tâches, les concepts de base peuvent également être mobilisés).</li>
    <li>Lorsque vous donnez une tâche aux élèves, ce qu’ils proposent n’est pas nécessairement identique à votre solution, mais leur idée peut néanmoins être pertinente.</li>
</ul>
</div>
