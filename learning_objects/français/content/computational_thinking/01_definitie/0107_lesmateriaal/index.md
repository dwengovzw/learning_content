# Matériel pédagogique

La pensée computationnelle peut s'inviter dans vos cours de multiples façons. Le programme existant offre de nombreuses possibilités pour se lancer dans la pensée computationnelle. 

- Des chercheurs du monde entier ont déjà beaucoup publié à ce sujet.
- Nous explicitons également autant que possible la pensée computationnelle présente dans notre matériel pédagogique.

Nous avons rassemblé ces exemples. Vous les retrouvez dans le parcours d'apprentissage ['Exemples'](https://www.dwengo.org/learning-path.html?hruid=ct3_voorbeelden&language=nl&te=true&source_page=%2Fcomputational_thinking%2F&source_title=%20Computationeel%20Denken#ct_inleiding_voorbeelden;nl;3). 

Des exemples concrets de leçons de pensée computationnelle sont développés dans le parcours d'apprentissage ['Études de cas'](https://www.dwengo.org/learning-path.html?hruid=ct6_cases&language=nl&te=true&source_page=%2Fcomputational_thinking%2F&source_title=%20Computationeel%20Denken#ct_cases0;nl;3). 

La courte vidéo ci-dessous explique comment Dwengo intègre la pensée computationnelle dans des projets STEM ou dans les cours de mathématiques.
