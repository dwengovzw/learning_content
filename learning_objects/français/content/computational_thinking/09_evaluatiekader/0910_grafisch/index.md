# Cadre d’évaluation

## Robot social : démarrer avec le simulateur Dwengo. Exercices de programmation simples visant l’apprentissage et la mise en pratique des principes et éléments de la programmation.  

**Objectif pédagogique :**<br>
Les élèves conçoivent un algorithme pour résoudre un problème numériquement.

**Prérequis :**
- Les élèves reformulent un problème de manière à ce qu’il soit possible de le résoudre à l’aide d’un ordinateur.<br>
- Les élèves conçoivent un algorithme en explicitant une série d’instructions ordonnées pour résoudre le problème.

**Objectifs d’apprentissage à évaluer :** 
- Les élèves convertissent l’algorithme en code graphique.
- Les élèves exécutent le code (ce qui se traduit par une simulation).
- Les élèves déboguent le code.

**Tâches pour les élèves :**
- Exercices de programmation.
- Débogage.
- Remplir le formulaire d’auto-réflexion.

**Schéma d’évaluation :**

![diagramme radar robot social](embed/spinsocialerobot.png)

**Tâches pour les élèves et forme d’évaluation proposée :**
- Exercices de programmation & Travail : évaluation du code (quantitative & formative).
- Débogage & Grille d’évaluation.
- Remplir le formulaire d’auto-réflexion & Auto-réflexion : après un exercice un peu plus important (qualitative & formative).
