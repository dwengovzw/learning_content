# Travailler avec des notebooks

**Enregistrer le notebook**

Vous pouvez télécharger le notebook au format *ipynb* pour continuer à y travailler ultérieurement. Pour cela, allez dans 'File' > 'Download'.<br>
Vous retrouverez ensuite le notebook dans le dossier Téléchargements de votre ordinateur. Vous pouvez ensuite placer le notebook dans un dossier où vous pourrez le retrouver facilement.

Vous pouvez enregistrer le notebook (exécuté) en *pdf* via le navigateur. Par exemple, dans Chrome : Imprimer > Enregistrer au format PDF. Il est recommandé de régler l'option 'Échelle' sur 85 pour cent.   

**Remarque : Il n'est pas nécessaire de se déconnecter, mais c'est possible. Le bouton de déconnexion se trouve dans le menu sous 'File', tout en bas.**  
*Auparavant, lorsque vous aviez terminé une série de notebooks, vous deviez vous déconnecter dans l'onglet du groupe de notebooks. Ce n'est donc plus nécessaire.*
