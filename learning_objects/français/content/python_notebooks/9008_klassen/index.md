# Classes
Python est un langage orienté objet. En Python, tout est un objet : chaque nombre, chaque chaîne de caractères... Les objets et méthodes connexes (fonctions spécifiques à ces objets) sont regroupés en classes. Dans ce notebook, tu apprendras à définir toi-même une classe.
