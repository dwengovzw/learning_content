# Le modèle SIR

Une épidémie est modélisée mathématiquement par le modèle SIR.  
SIR signifie *Susceptible* (vulnérable), *Infected* (infecté) et *Resistant* (résistant ou rétabli), les trois types d’individus présents dans une communauté. 
La plupart du temps, les individus sont simplement des humains, mais ce modèle peut également être utilisé pour modéliser des flambées de maladies chez les rongeurs, les oiseaux ou même les plantes. 

Grâce au modèle SIR, on simule par exemple comment le nombre d’infections évoluera ou quel est l’effet de la vaccination.
