# Introduction
En troisième secondaire, le théorème de Pythagore est abordé. Les exercices qui l’accompagnent se prêtent à travailler avec quelques concepts de la pensée computationnelle et de la programmation : entrée-traitement-sortie, reconnaissance de motifs, fonctions ...

Ce qui est donné au sujet des côtés des triangles peut être demandé à l’utilisateur. Au moyen de code, cette _entrée (input)_ peut alors être _traitée_ automatiquement et suivie d’une _sortie (output)_ qui est affichée.

Le nombre de types d’exercices liés au théorème de Pythagore en troisième secondaire est assez limité. C’est pourquoi le code nécessaire dans les exercices est souvent le même. Il est donc utile d’encapsuler ce code dans une _fonction_ qui pourra ensuite être réutilisée dans les différents exercices. De plus, travailler avec des fonctions rend les programmes plus lisibles.
Identifier le type de _problème_ à résoudre relève de la _reconnaissance de motifs_.

---
Ce parcours d’apprentissage traite d’abord de la manière de se lancer avec entrée-traitement-sortie et de la façon de définir soi-même une fonction en Python. Ensuite, tu utiliseras ces connaissances pour résoudre des problèmes dans le cadre du théorème de Pythagore.
