Dans le parcours d'apprentissage « Régression linéaire », tu as appris à utiliser Python pour trouver la droite qui s'ajuste le mieux à un nuage de points donné.
Tu trouveras ici un notebook qui résume la démarche. <br>Tu peux utiliser ce notebook pour rechercher la droite de meilleure adéquation, mais, moyennant les adaptations nécessaires, le code du notebook convient aussi pour *ajuster* d'autres courbes à un jeu de données.
