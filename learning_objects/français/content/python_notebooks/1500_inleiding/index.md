# Introduction
Une image numérique se compose d'un nombre fini de pixels. Si vous placez suffisamment de petits carrés dans une grille, vous pouvez représenter une grande figure.<br>
Le mot « pixel » est dérivé des mots anglais *picture* (image) et *element*.<br>
Le mot « numérique » vient du latin *digitus* (doigt) et fait référence au comptage sur les doigts. 

Sur une bonne photo, on ne distingue pas les pixels, sauf si l'on zoome fortement sur l'image.  

![](embed/begoniazoom.png "Photo zoomée")

Dans ce parcours d'apprentissage, vous apprendrez comment ajuster et manipuler les couleurs d'une image.
