Voulez-vous savoir si la régression linéaire est pertinente pour un jeu de données donné ? La visualisation des données peut y contribuer.  
Cependant, le degré de corrélation ne peut pas toujours être estimé à l'œil. Vous apprendrez pourquoi la standardisation des données est importante à cet égard.
