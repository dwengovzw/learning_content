# Introduction

Travailler avec des diagrammes de dispersion est l’un des objectifs d’apprentissage du deuxième degré. Dans cette série de notebooks, tu apprends en Python à réaliser un diagramme de dispersion à partir d’un jeu de données à deux variables. Tu vas tracer les données dans un système d’axes, le résultat est un nuage de points ou un diagramme de dispersion. Il peut s’agir de données que tu saisis toi-même ou de données que tu charges depuis un fichier. 
Avec deux autres notebooks, tu peux t’exercer à ce que tu as appris. L’importance de la visualisation apparaît clairement dans deux autres notebooks.

De plus, ce parcours d’apprentissage constitue la base de tous les parcours dans lesquels tu traces des graphiques. Les notebooks indiquent également quels concepts de la pensée computationnelle et quels principes de la programmation sont abordés.
