# Introduction

La régression linéaire est un objectif d'apprentissage pour le deuxième degré. Dans ces notebooks, tu apprendras avec Python à trouver et à visualiser la droite qui s'ajuste le mieux à un jeu de données, en utilisant des données 'réelles'. Parfois, cette droite reflète une tendance contenue dans les données; on parle alors d'une ligne de tendance.
