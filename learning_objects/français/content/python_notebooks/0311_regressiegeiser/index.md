# Application
## Geyser
Tu as déjà travaillé avec les données du geyser Old Faithful dans un notebook du parcours d'apprentissage ['Diagramme de dispersion'](https://www.dwengo.org/learning-path.html?hruid=maths_spreidingsdiagrammen&language=nl&te=true&source_page=%2Fstem%2F&source_title=%20STEM#pn_inleiding_spreidingsdiagram;nl;3).

Un geyser est une source d'eau chaude chauffée naturellement qui projette régulièrement de l'eau chaude et de la vapeur dans l'air.
Dans ce notebook, tu étudies la relation entre la durée des éruptions (jaillissements) et le temps qui s'écoule jusqu'à la prochaine éruption.
