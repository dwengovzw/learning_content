# Introduction

Dans ces notebooks STEM, tu vas travailler avec des données qui rendent le changement climatique concret : des données sur la concentration atmosphérique de CO<sub>2</sub> au fil des siècles, des données sur l'épaisseur ou la longueur de deux glaciers en Suisse, des données sur le niveau de la mer à Ostende. Tu représenteras les données dans un diagramme de dispersion et tu recevras également quelques exercices qui t'aideront à interpréter les données. Si tu le souhaites, tu peux aussi exploiter les données pour étudier des évolutions prospectives.
