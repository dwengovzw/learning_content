# Du JPEG au NPY

Dans le premier notebook, un exemple montre comment convertir un fichier JPG en une image en niveaux de gris, représentée par une matrice, et comment enregistrer cette image sous forme de fichier NPY. Pour cela, vous utilisez le module scikit-image (skimage), qui fournit des algorithmes pour le traitement d’images numériques et la vision par ordinateur. 

Dans le deuxième notebook, vous découvrez l’ensemble de données MNIST et appliquez ce que vous avez appris dans le premier notebook.
