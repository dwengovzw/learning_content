# Mondrian

Avec des graphes, on peut présenter certaines données autrement. On peut p. ex. représenter les couleurs d’un tableau de Mondrian avec leurs couleurs voisines au moyen d’un graphe.

![dd-2019 09 23-12](https://user-images.githubusercontent.com/48352335/214017889-150dd75e-87b2-460b-955d-897539e03834.jpg)

Dans la présentation ci-dessous, on voit comment procéder.

![](@youtube/https://www.youtube.com/embed/x0GSC5t7HoI "Comparer des graphes - Couleurs")
