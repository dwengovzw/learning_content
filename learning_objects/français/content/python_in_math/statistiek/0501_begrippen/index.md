# Quelques concepts

## Fréquence absolue et relative

Dans cette partie du parcours d'apprentissage, vous faites vos premiers pas dans l'utilisation de Python pour de grands jeux de données.

Dans un premier notebook, vous apprenez à importer des jeux de données stockés en externe et comment obtenir avec Python des notions de base telles que la fréquence absolue et relative, ainsi que la taille d’un jeu de données.<br>
Dans un deuxième notebook, vous appliquez ce que vous avez appris à un jeu de données réel.
