# Distribution normale

Une **variable aléatoire** est une variable dont la valeur dépend du hasard.<br>
Une **loi de probabilité** contient toutes les issues possibles d’une variable aléatoire avec, à chaque fois, la probabilité correspondante qu’une issue se produise. 

Les variables aléatoires, telles que la taille, le poids, le QI, la masse d’un paquet de sucre, les résultats d’une expérience de physique et les scores à l’Olympiade de mathématiques, sont distribuées de manière *continue*. Elles peuvent prendre toutes les valeurs réelles à l’intérieur d’un certain intervalle. Le graphique de cette distribution continue présente en outre certaines caractéristiques; c’est une **distribution normale**.<br>
Les valeurs qu’une telle variable aléatoire prend sont symétriques par rapport à la valeur moyenne. Une grande partie des valeurs ne s’écartera pas beaucoup de la moyenne, tandis que des valeurs extrêmement grandes ou extrêmement petites apparaissent rarement.<br> 
Le graphique de la distribution normale est ce qu’on appelle une **courbe de Gauss** et a la forme d’une *cloche*.<br>
La fonction correspondante s’appelle une **fonction de répartition** ou fonction de densité de probabilité.

Dans le premier notebook, tu apprendras avec Python à tracer les graphiques des distributions normale et normale standard. Tu y apprendras aussi comment calculer des probabilités avec Python dans ce contexte.<br>
Dans le second notebook, tu appliqueras ce que tu as appris à un grand jeu de données du monde réel.
