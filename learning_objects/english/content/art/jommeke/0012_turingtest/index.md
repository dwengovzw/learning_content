## Turing test

As early as 1936, Alan Turing devised a test to determine the extent to which a computer system exhibits human intelligence. In the original test, a user must chat with a person they do not know. The person on the other end of the chat is either a real person or a computer program trying to imitate a person. If the user is chatting with a computer but does not realize it, then the computer program passes the Turing test.
