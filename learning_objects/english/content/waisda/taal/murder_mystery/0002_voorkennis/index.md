# Prerequisites

For this activity, it can be useful to have some programming knowledge. You will be working with Python here, but if you have experience with another programming language, that should be sufficient as well. If you want to be sure that you have sufficient Python knowledge for this activity, you can independently go through [the course material on Python](https://dwengo.org/python_programming/).

Furthermore, for this activity you will use Jupyter Notebook. This is a simple online platform that lets you run Python code for free, and without installing anything, in an interactive Python notebook. If you want to know how to work with such a notebook, you can go [here on the website](https://dwengo.org/learning-path.html?hruid=pn_werking&language=nl&te=true&source_page=%2Fkiks%2F&source_title=%20KIKS#pn_werkingnotebooks;nl;3).
