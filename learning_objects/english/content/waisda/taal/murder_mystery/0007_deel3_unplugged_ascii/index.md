# Part 3: ASCII

- In the first box, the participants also found an ASCII table. It shows which numbers the computer uses to represent letters.
- At the end of the notebook in 'Part 2' they receive the name of the prime suspect and a hint to the ASCII table.
- By converting the letters in the prime suspect's name to numbers and adding them up, they obtain the code for box 2.
