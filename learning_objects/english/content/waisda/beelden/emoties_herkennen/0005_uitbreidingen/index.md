# Extensions 

* You now have a system that can distinguish between two emotions. You can easily extend this system to three emotions. To do so, you first need to collect additional data. Once you have this data, you can use the code in the notebook *emoties.ipynb* (from the 'automatic detection' section of this learning path) to build a new system that can distinguish between three emotions.

* In principle, you can use the same system to distinguish any two objects from each other, for example cats and dogs. In that case, instead of images of smileys, you would need to make drawings of cats and dogs.
