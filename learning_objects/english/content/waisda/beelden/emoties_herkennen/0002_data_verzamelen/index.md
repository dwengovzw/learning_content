# Collecting data

Before you can train an AI system, you need a high-quality dataset. In this activity, participants are responsible for compiling this dataset themselves. They do this using the following grid.

![](images/raster.svg)

You can download the PDF of the grid [here](images/raster.pdf). Print this grid at least 10 times in A3 format. This way, participants can collect enough data to train their own AI system.

You start by building a simple AI system that can distinguish between the emotions *happy* and *surprised*. For each of these emotions, you need about 100 examples. To collect these examples, participants can draw smileys in the grid with the corresponding emotion. Below you can see two examples of completed grids.

![](images/voorbeeld_raster_blij.jpg)
![](images/voorbeeld_raster_verbaasd.jpg)

**Note**: Make sure that a grid contains only drawings of one type of emotion. A grid must therefore not contain both happy and surprised smileys at the same time.
