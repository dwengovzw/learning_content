# Debriefing

To ensure sufficient learning effect among the participants, it is important to discuss the activity with them afterwards.

* Have someone from each group come forward in turn.
* Do the drawings match?
* Have a student explain the chosen protocol.
* Some possible questions you can ask:
    * How would you change your protocol if the size of the grid is unknown?
    * How many numbers would you send if you were sending a white image or a completely black image?
    * How could you reduce the number of transmitted numbers?
    * What is the effect of errors in the transmitted numbers on the final image?

If you want to print a more detailed description of this activity, you can download the [activity sheet](content/menselijkComputernetwerk.pdf).
