## Alternative versions

If your school cannot or does not want to work with Google products, there are alternative versions of Teachable Machine. For example:

* [In-browser version of the Teachable Machine developed within the European Generation AI project.](https://tm.gen-ai.fi/image/general)
* [the Teachable Machine in the VRT EDUbox AI](https://edubox.vrtnws.be/);
* [your own version of Teachable Machine](https://github.com/googlecreativelab/teachable-machine-v1?tab=readme-ov-file) that you install via the source code on GitHub;
* [a similar tool LerningML](https://web.learningml.org/en/home/).
