# Building a Drawing Robot

Now that you have an idea of how a robot works, you can start building the robot itself! Use the [photo](images/Voorbeeld.pdf "finished robot") and the parts from the box.

![](images/Voorbeeld.png "finished robot")

Can you manage to build the robot completely, just by looking at the photo? Do you need a bit of extra help? Then watch the video below.
