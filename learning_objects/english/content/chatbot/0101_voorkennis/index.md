# Prerequisites

This learning path contains a Python notebook that illustrates how a simple knowledge-based chatbot works. 

To be able to work through this notebook, the following learning path is required as a prerequisite:

* Learning path ['Working with notebooks'](https://www.dwengo.org/learning-path.html?hruid=pn_werking&language=nl&te=true&source_page=%2Fchatbot%2F&source_title=%20Chatbot#pn_werkingnotebooks;nl;3)

If you also want to properly understand the details of the code that appears in it, please also go through the following learning paths:

* *'Programming with Python'* > Learning path *'Structures'* > Section *'Loop structure'* > Item 1. of the notebook
* *'Programming with Python'* > Learning path *'Data types'* > Sections *'Language technology - String'* and *'Language technology - Dict'*
