# Introduction
Chatbots are software systems that can interact with people via **typed text messages**. This is done through spontaneous text in natural language. A chatbot is therefore nothing more than a robot you can chat with, an automated digital system with which you can engage in dialogue. Modern chatbots are an application of Natural Language Processing (NLP). 

In this learning path, you will become familiar with the workings of these systems.
