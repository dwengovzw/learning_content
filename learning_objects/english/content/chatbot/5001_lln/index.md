# What is a chatbot?

Chatbots are software systems that can interact with people via typed text messages. **A chatbot is nothing more than a robot you can chat with.**

## Types of chatbots

There are rule-based chatbots, such as Billy from bol.com, and there are learning chatbots, such as ChatGPT.

## Assignment

Billy, the bol.com chatbot, is a rule-based chatbot.

Have a chat with Billy. You can find him at the bottom left of this [web page](https://www.bol.com/nl/nl/m/klantenservice/). <br>
Can you confuse him?
