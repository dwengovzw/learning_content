# dwenguinoBlockly
## Uploading to the Dwenguino

The video below shows how you can upload a program from any computer and browser to the Dwenguino.
It is important that you follow all the steps as shown in the video!

* Write the program
* Download the program using the button ![alt](embed/menu_uploaddwenguino.png "download menu")
* Open Downloads; there is now a .dw file
* Connect the Dwenguino and press RESET + SOUTH, then release the reset button
* The Dwenguino appears among your folders as a USB drive
* Copy the .dw file to the Dwenguino
* Press RESET
